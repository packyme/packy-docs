登录入口：<https://www.packyapi.com/login>

![](/assets/image/QuickStart/002.webp)

## 使用 Google 账号登录
1. 点击 “使用 Google 继续”
2. 选择注册时使用的 Google 账号
3. 授权后即可自动登录

## 使用邮箱/用户名登录
1. 点击 “使用邮箱或用户名登录”
2. 输入邮箱地址或用户名
3. 输入密码
4. 点击登录

::: info 设备登录说明
浏览器会保持登录状态；在新设备需重复登录流程。
:::
